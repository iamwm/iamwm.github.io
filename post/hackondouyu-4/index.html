原本打算使用selenium给这个小项目做自动化测试的，没成想selenium功能如此强大，那就顺势做个自动抢鱼丸的小工具吧。😊

Selenium是什么

Selenium是一款由ThoughtWorks公司开发的web自动化测试工具，它可以让测试人员编写的测试代码直接在浏览器中进行测试，并且它提供了当前绝大多数现代浏览器的驱动。同时Selenium也拥有很多开发语言版本的驱动。我使用的就是python版本的驱动。具体安装步骤可以参考这里。 具体来说，Selenium可以模拟大多数的页面操作：填写表单、拖拽页面、页面切换、返回上一页面、前进到下一页面等。 告诉了Selenium网址之后，它可以自动打开指定的浏览器，并且待页面完全加载之后分别通过页面元素的id、name、XPath、Link Text、tag、class或者是css选择器完成元素的选择。 选定了页面元素之后可以按照预期完成一些列动作。

Talk is cheap, Show you my code

整个项目的代码在github上托管，感兴趣的同学可以前去浏览。整个抢鱼丸礼物的过程可以由这个图来说明：

具体步骤如下：

  1. 与斗鱼弹幕服务器建立连接，监听当前最热门房间弹幕，由于火箭礼物消息会通过广播向所有直播间发送，所以只需要根据火箭消息的标识符来判定火箭消息即可；
  2. 火箭消息接收之后可以将其转存到redis中，利用pub/sub完成火箭消息的广播和订阅；
  3. 订阅redis火箭消息频道，获取当前可以抢鱼丸的房间完整url；
  4. 利用selenium打开浏览器并访问指定直播间，自动发送弹幕消息，并且点击抢鱼丸的按钮，最终完成自动抢鱼丸礼物的整个流程。

def get_rocket(data):
    try:
        sender_id = re.search('\/sn@=(.+?)\/', data).group(1)
        recver_id = re.search('\/dn@=(.+?)\/', data).group(1)
        recver_room = re.search('\/drid@=(.+?)\/', data).group(1)
        gift = re.search('\/gn@=(.+?)\/', data).group(1)
        rocketmsg = {}
        rocketmsg["sender_id"] = sender_id
        rocketmsg["recver_id"] = recver_id
        rocketmsg["recver_room"] = recver_room
        rocketmsg["gift"] = gift
        rocketmsg["date"] = datetime.now()
        col.insert_one(rocketmsg, bypass_document_validation=False)
        publishvalue = {}
        publishvalue["sender_id"] = sender_id
        publishvalue["recver_id"] = recver_id
        publishvalue["recver_room"] = recver_room
        publishvalue["gift"] = gift
        castRocket(publishvalue)

            gift, "<", datetime.now(), ">"
        # 在此处完成自动抢鱼丸流程
        getGift(str(recver_room))
    except Exception, e:
        print "error occur:", repr(data)
    finally:
        pass

from selenium import webdriver
from selenium.webdriver.common.action_chains import ActionChains
import time


def getGift(roomid):
	# 这里是指定firefox的cookie位置
    fp = webdriver.FirefoxProfile(
        r'/Users/eclipse/Library/Application Support/Firefox/Profiles/tmsbsjpg.default')
    browser = webdriver.Firefox(fp)
    browser.implicitly_wait(15)  # seconds
    browser.get("http://www.douyu.com/" + roomid)
    try:
    	# 找到弹幕聊天窗口，输入聊天内容
        indexvideo = browser.find_element_by_class_name('cs-textarea')
        print type(indexvideo)
        indexvideo.send_keys('2333333333333')
        print indexvideo
        time.sleep(7)
        sendbut = browser.find_element_by_class_name('b-btn')
        ActionChains(browser).move_to_element(
            indexvideo).click(sendbut).perform()
        gift = browser.find_element_by_class_name('peck-cdn')
    except Exception, e:
        print str(e)
        browser.quit()

    times = 0
    while True:
        try:
        	# 点击抢鱼丸按钮，在完成之后，按钮消息，继续点击出现异常，自动关闭浏览器即可
            ActionChains(browser).move_to_element(gift).click(gift).perform()
            time.sleep(2)
            print times
            times += 1
        except Exception, e:
            print 'completed by an error'
            browser.quit()
